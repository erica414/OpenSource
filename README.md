# 구글 리뷰 데이터를 이용한 부산 로컬 국밥 맛집 찾기

## 🔑 프로젝트 소개
> 구글 리뷰 데이터 기반 사용자 선호도 맞춤 '실패 없는' 부산 국밥 맛집 추천 데스크톱 GUI 애플리케이션

## 💡 계획 의도
부산은 국밥집이 너무 다양하고 넓게 퍼져 있어 관광객이 식당을 선택하기 어렵습니다.
그래서 구글 리뷰 데이터를 키워드 기반으로 분석하여, 사용자가 원하는 특징(맛, 서비스 등)을 입력하면 그에 맞는 식당을 추천해주는 시스템으로 개발된 프로젝트입니다.

## 🚀 사용법 (How to Use)

### 1. 선호 키워드 입력
- 프로그램을 실행하고, '맑은 국물', '고기 양', '주차' 등 원하는 국밥집의 특징을 키워드로 선택합니다.

(GUI 실행 화면 예시 사진)

### 2. 맞춤 맛집 추천
- '추천' 버튼을 누르면, 선택한 키워드를 기반으로 가장 일치하는 부산 국밥 맛집 목록과 추천 사유(키워드 지표)를 보여줍니다.

(추천 결과 화면 예시 사진)
